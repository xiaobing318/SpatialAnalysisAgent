# 说明
 - 这个文档的作用就是为了解释说明当前目录install_packages中各个文件的作用是什么
 - requirements.txt
   - 文件中指定该项目python使用哪些包
   - 罗列出当前python使用到了哪些包
 - py3-env.bat
   - 设置环境变量
   - 调用对应路径下的.bat文件
 - install_pip_packages.bat
   - 调用py3-env.bat
   - 安装requirements.txt文件中指定的python packages
 - check_packages.py
   - 检查python packages
